2. 선택자(selector)
    1) 전체 선택자
        - 모든 HTML 요소를 한꺼번에 선택하기
    2) 태그 선택자
        - 태그명을 이용하여 선택하기
    3) 자손 선택자
        - 자식과 하위 요소를 모두 선택하기
    4) 자식 선택자
        - 직계 자식 요소만 선택하기 ( > )
    5) 그룹 선택자
        - 여러 개의 요소를 그룹으로 선택하기 ( , )
